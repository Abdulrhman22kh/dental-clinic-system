## ✨ Key Features

- **📊 Real-Time KPI Dashboard:** Instant tracking of total appointments, pending patient queues, and expected revenue.
- **📅 Advanced Appointment Management:** Seamless recording of patient details, custom treatment types (Fillings, Cleaning, Extractions, Braces, Root Canal), and timestamps.
- **🔄 Interactive Status Workflow:** Toggle appointment states dynamically between *Waiting* and *Completed* with instant visual feedback.
- **🔍 Live Search Filtering:** Instant name-based filtering across all clinic records.
- **🌙 Dark / Light Mode Support:** Fully integrated theme switcher with persistence via browser `LocalStorage`.
- **💾 Smart Data Persistence:** Zero data loss on refresh thanks to secure client-side storage handling.
